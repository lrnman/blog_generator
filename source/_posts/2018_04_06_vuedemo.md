使用vue2.5+vue-router3.0+vuex3.0

完整的登录、退出、页面验证是否需要登录。

为了专注vue上，对后端的api操作是用异步模拟的,如果你有服务器，可以去重写api.js。

1、下载项目，进入项目文件夹   
2、

```
npm i
```
3、

```
npm start
```
4、打开浏览器，输入 http://localhost:8080
